ORBITAL GUARDIAN
LIVE CELESTRAK DATA · REAL PYTHON scikit-learn ML · PYODIDE WASM · TEAM LEGION · ASTRAVA 2024
🌐
LIVE DATA
Fetches real TLE data directly from CelesTrak in real time — the same source used by NASA and every space agency. Positions are computed from current orbital parameters, not cached data.
🐍
REAL PYTHON ML
scikit-learn Isolation Forest (n_estimators=200) and Random Forest (n_estimators=300) run inside the browser via Pyodide WebAssembly. Same code as a backend server, zero server needed.
🛰️
SGP4 TRACKING
Satellite positions computed with SGP4 — NASA/NORAD standard algorithm used to track 27,000+ orbital objects daily. Accurate to sub-kilometer in LEO.
⚠️
COLLISION AI
3D ECI closest-approach analysis across all satellite pairs. Random Forest ML gives collision probability from distance, relative speed, and altitude difference.
🤖
ANOMALY DETECTION
Isolation Forest trains on live orbital data every session. Flags satellites with abnormal altitude or velocity without needing any labelled training data.
💀
KESSLER SYNDROME
One collision creates debris, debris causes more collisions. We detect the cascade precursors in real time before LEO becomes unusable for decades.
